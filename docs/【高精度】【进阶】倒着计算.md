# 倒着计算乘法
```cpp
void times(int *a, int num)
{
    // 乘法计算，首先算每一位的乘法
    for (int i = 0; i < N - 1; i++)
    {
        // 数组的下标到了N-1就到头了，因为要预留一位，所以是N-2就退出循环
        a[i] *= num;
    }
    for (int i = 0; i < N; i++)
    {
        // 这里要注意，这里的进位是往数组扩充的方向进行计算的。
        a[i + 1] += (a[i] / 10);
        a[i] %= 10;
    }
}
```

# 除法
```cpp
void div(int *aa, int *bb, int num)
{                              // 除法
    memset(bb, 0, sizeof(bb)); // 赋为0
    int x = 0;
    for (int i = N - 1; i >= 0; i--)
    {                       // 从高位到低位
        x = x * 10 + aa[i]; // 挪位再加
        bb[i] = x / num;    // 记录
        x %= num;           // 模上
    }
}
```

# 倒着比较
```cpp
bool more(int *aa, int *bb)
{ // 比较, 从最右侧，也就是反向数组的最高位开始比较
    for (int i = N - 1; i >= 0; i--)
    {
        if (aa[i] > bb[i])
            return 1;
        if (aa[i] < bb[i])
            return 0;
    }
    return 0; // 注意这里也要写上，写0写1随便
}
```
